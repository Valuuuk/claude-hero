MCP-конектори: різниця з плагінами і як підключити Gmail, Notion, Telegram

MCP — це не "плагіни 2.0". Це принципово інша архітектура. Розбираю, чому вона змінює гру і як її використати.

Що таке MCP

MCP — Model Context Protocol. Це стандарт, через який Claude (і не тільки Claude — ChatGPT, Gemini теж його підтримують) спілкується із зовнішніми інструментами: твоєю поштою, календарем, базою знань, Telegram, Linear, Figma, чим завгодно.

Простіше: MCP — як USB. Раніше кожен пристрій мав свій кабель. Тепер усі говорять однією мовою.

Чим це відрізняється від плагінів

  • Плагіни жили всередині самого Claude. Кожен сервіс мав сам зробити свій плагін під Anthropic. Це повільно і обмежено.
  • MCP-сервери — це окремі програми, які може запустити будь-хто. Вони відкриті, є тисячі готових, і ти можеш написати свій за вечір.
Висновок: якщо плагіни — це "магазин додатків Apple", то MCP — це Linux. Більше свободи, більше можливостей, трохи більше зусиль на старті.

Готові офіційні конектори

В інтерфейсі Claude (Pro+) у налаштуваннях є вкладка Connectors. Там одним кліком підключаються:

  • Gmail — читання і відправка пошти
  • Google Calendar — створення подій, перевірка вільних слотів
  • Google Drive — пошук і редагування документів
  • Notion — пошук, створення сторінок, оновлення баз даних
  • Linear — створення issue, керування проєктами

Сторонні MCP — Telegram, Slack, банки

Готові open-source MCP-сервери є для майже всього. Декілька, які я ставив сам:

  • Telegram MCP — Claude читає твої чати, шукає, відправляє повідомлення
  • iMessage MCP — те саме для Apple-повідомлень (тільки macOS)
  • TickTick MCP — таски і списки
  • Apple Notes MCP — читання і запис у Нотатки
  • Computer Use MCP — Claude керує мишею і клавіатурою на твоєму компі

Як підключити перший MCP — за 3 хвилини

  1. Відкрий Claude Desktop → Settings → Connectors
  2. Знайди потрібний сервіс у списку, натисни "Connect"
  3. Авторизуйся в браузері (OAuth)
  4. Готово — у новому чаті Claude бачить твою пошту/календар/Notion

Перевір, попросивши: "Що в мене заплановано на завтра?" або "Знайди мою останню переписку з [ім'я]".

Що далі

  • Cowork — Claude з кількома MCP в режимі "паралельні агенти"
  • Skills — як скіли і MCP працюють разом
